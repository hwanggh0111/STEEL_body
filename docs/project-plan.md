# IRON LOG 프로젝트 플랜

## 기술 스택
- Frontend: React 18 + Vite + Zustand + Recharts
- Backend: Node.js + Express + SQLite
- 스타일: CSS Variables (디자인 시스템 기반)
- 인증: JWT

## 기능 목록
- [ ] 로그인 / 회원가입
- [ ] 운동 루틴 추천 (머신 / 맨몸 / 부위별)
- [ ] 운동 기록 CRUD
- [ ] 인바디 기록 CRUD + BMI 계산
- [ ] 헬스 장비 추천 (카테고리 필터)
- [ ] wger.de API 운동 검색
- [ ] 히스토리 (날짜별 조회)
- [ ] 대시보드 (차트)

## 작업 순서
1. 디자이너: 디자인 시스템
2. 백엔드: API 서버 + DB
   (1, 2 동시 진행 가능)
3. 프론트엔드: React 앱
4. QA: 검토 + 버그 리포트
